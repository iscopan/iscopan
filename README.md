## 📂 Proyectos Destacados

### 🔥 [Emojournal](https://emojournal.app)
> Aplicación social para expresar emociones con emojis

Una plataforma que permite a los usuarios registrar sus emociones diarias de forma privada o compartirlas con amigos mediante un sistema de solicitudes de amistad. Backend serverless con Firebase.

**Stack:** Flutter, Firebase, TypeScript, Jest  
⭐ 3 estrellas

---

### 🏥 [Nefroamigo](https://nefroamigo.fpuente.com)
> PWA para gestión de salud renal con enfoque privacy-first

Aplicación diseñada para pacientes con enfermedad renal crónica y transplantados. Gestiona medicación compleja, hidratación, presión arterial y peso. Destaca por su arquitectura de privacidad: **cero datos en servidor**, todo se almacena localmente e implementa notificaciones push.

**Stack:** React 19, Vite, Tailwind CSS v4, PWA, Framer Motion  

---

### 🎮 Muxaxos
> Aplicación de escritorio para crear y presentar concursos

Herramienta profesional para crear sesiones de concurso tipo *Jeopardy* con soporte multimedia completo. Incluye panel de categorías, quizzes con opciones múltiples, y modo de presentación en segunda pantalla.

**Stack:** React 19, TypeScript, Tauri v2, Rust
